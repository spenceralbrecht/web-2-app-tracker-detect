# Tracker Detect Pro (Clario Replica)

This project is a web-based replica of the Clario onboarding flow, built using **Vite** and **Vanilla JavaScript**.

## Tech Stack

-   **Framework**: None (Vanilla JS)
-   **Build Tool**: Vite
-   **Styling**: Vanilla CSS (with CSS variables for theming)

## How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

4.  **Preview Production Build**:
    ```bash
    npm run preview
    ```

## Project Structure

-   `src/main.js`: Main application logic and state management.
-   `src/data.js`: Configuration for the questionnaire steps.
-   `src/style.css`: Global styles and theme variables.
-   `src/*.css`: Component-specific styles (carousel, news, graph).
-   `public/`: Static assets (images, icons).
